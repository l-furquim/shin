# Shin Video Frontend

## Upload (index.html)
Simple HTML/CSS/JS frontend for testing chunked video uploads.

## Player (player.html)
DASH video player using dash.js for playing adaptive bitrate video from CloudFront CDN.

Usage:
1. Enter CloudFront URL (e.g., https://d123456.cloudfront.net)
2. Enter Video ID
3. Select resolution
4. Click Load Video

The player supports adaptive bitrate streaming and displays real-time quality and buffer metrics.
